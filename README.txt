HOW TO PUT THIS LIVE ON THE INTERNET (FREE)
==================================================

STEP 1 — Create a GitHub account (free)
  → Go to github.com and sign up

STEP 2 — Create a new repository
  → Click "New Repository"
  → Name it: marketpulse
  → Set it to Public
  → Click "Create repository"

STEP 3 — Upload all these files
  → Click "uploading an existing file"
  → Drag and drop ALL files from this folder
  → Click "Commit changes"

STEP 4 — Deploy on Vercel (free hosting)
  → Go to vercel.com
  → Sign up with your GitHub account
  → Click "New Project"
  → Import your "marketpulse" repository
  → Click "Deploy"
  → Your website is LIVE in 2 minutes!

STEP 5 — Add your Anthropic API Key
  → In Vercel dashboard, go to Settings → Environment Variables
  → This is needed for the screener to work
  → Get your API key from: console.anthropic.com
